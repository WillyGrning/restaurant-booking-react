# Restaurant Table Booking App

Frontend-only restaurant table booking application built with React + Vite.

## Features
- Book a table with date, time, and guest count
- Booking confirmation page
- My Booking page
- Dynamic navbar based on booking state
- LocalStorage persistence
- No backend (frontend architecture focus)

## Tech Stack
- React 19
- Vite
- React Router
- Context API
- localStorage
